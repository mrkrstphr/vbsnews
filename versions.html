vbsNews: Version History
This file contains information about release versions of vbsNews and the major updates included within each update.

Version 0.62 - Released June 28th, 2004
- Fixed duplicate user bug
- Fixed blank category bug
- Fixed "seeking" bug in menu.php
- Updated tag interface for news posting (cross-browser)
- Better list tag handling in news display
- Implemented customizable subject lengths (up to 256 chars)
- Category restrictions for archives, search, etc
- Individual category templates
- Revamped Modify News Section
- Added Page Links template

Version 0.59.4 - Released November 6th, 2003
- Fixes the tblPrefix bugs

Version 0.59.2 - Released October 24th, 2003
- Fixes category bug in manage categories
- Fixes modify news bug for regular users
- Enables MySQL error functionality in config.php

Version 0.59 - Released October 19th, 2003
- Guest commenting system implemented
- Categorical posting implemented
- Limit user's category options
- Template administration (no more uploading)
- More templates for more control (mostly comments)
- Hide email and reset password options for users
- Better tag interface in posting (double for IE users)
- Name storage now by ID for quicker handling
- E-mail confirmations for passwords (signup and change)
- Better Support (because it hasn't existed before)
- Table prefix option in CONFIG
- Date/Time format enhancements
- List tag bug eliminated
- Mozilla form problem corrected
- Known coding redundancies/SQL calls removed
- Fixed 'long word' bug

Version 0.47 - Released July 17th, 2003
- First stable release
- Fixed Mozilla submit bug
- Addressed security issues with .inc files
- Fixed all other known bugs

Version 0.43b - Released June 9th, 2003
- Fixed slashes bug
- Implemented password encryption
- Fixed install.php bugs
- Implemented JavaScript form validation
- Fixed news display bug
- Fixed the modify news bug

Version 0.41b - Released June 6th, 2003
- Rewrote modify news
- Added searching capability
- Rewrote template system
- Created vbsCode support in form sections
- Added HTML code support (via config variable)
- Added [img] tag support (can be shut off)
- Added database server support in $CONFIG
- Removed duplicate doerror() function in favor of do_error()
- Created create_form() function for submit and modify
- Fixed no subject bug

Version 0.35b - Released June 1st, 2003
This version of vbsNews was release to fix two important bugs with the previous release.

- Fixed the apostraphe bug in index.php
- Fixed the "function already defined" error in display.php

Version 0.32b - Released March 21st, 2003
- Original release
